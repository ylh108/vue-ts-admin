# vue-admin-ts
vue,cli3.0,typescript,后台管理系统模板

## Project setup
```
yarn install (因jspdf不支持yarn命令安装，改为npm install)
```

### Compiles and hot-reloads for development
```
yarn run serve 
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
